HTTP性能测试工具wrk

发表于 2018-07-23 |

简介

wrk是一个HTTP性能测试工具，在单个多核CPU上运行时能够产生大量负载。它结合了多线程设计和可扩展的事件通知系统，如epoll和kqueue。

可选的LuaJIT脚本可以执行HTTP请求生成，响应处理和自定义报告。可以在SCRIPTING中看到详情，scripts/目录中也有一些示例。
