Java中使用LinkedHashMap实现LRU缓存

发表于 2017-04-05 |

LRU简介

LRU是最流行的缓存算法之一。

LRU是Least Recently Used的缩写，意思是“最近最少使用”。

简单的说，LRU缓存一定量的数据，超过设定的阀值，就把一些过期数据删除掉。LRU会保留最近被使用过的数据，优先删除最久未被使用过的数据。

使用Java中的LinkedHashMap，可以快速实现一个LRU缓存。

LinkedHashMap的两种数据排序方式

  • insertion order（插入顺序排序），LinkedHashMap排序的默认值
  • access order（访问顺序排序）
    • LRU缓存会使用LinkedHashMap的access order
    • LinkedHashMap有一个removeEldestEntry方法，决定是否要移除最老的数据
